# Client-Server_Chatroom
Ky repository përmban kodin burimor të aplikacionit për komunikim klient-server. 
Serveri shërben si ndërmjetësues mes klientëve, për të mundësuar shkëmbimin e mesazheve mes tyre. 
Serveri dhe klienti para se të krijojnë lidhjen mes tyre, ata duhet të identifikojnë njëri-tjetrin përmes certifikatave. 

##Për të ekzekutuar aplikacionin:
Duhet të siguroheni që e keni të instaluar python në sistemin tuaj. 
Shkarkoni ose klononi repository-n
Për të startuar serverin: py server.py
Për të startuar klientin: py client.py

### Veglat e përdorura
* Python 3.6
* Visual Studio 2019
* Librari të gatshme në Python
